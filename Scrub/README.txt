Here is the skeleton version of the project. It contains the framework to create a portfolio with different content. 

General Information:

Files
-----
README.md : Contains some markdown syntax for getting started
_config.yml : Implements the Jekyll theme as well as some global content, i.e. email, phone number
index.md: This is the home page for the website

Folders
-------
Projects: Contains a markdown dedicated to each project to be showcased
Images: Contains all of the image files used throughout the website
Resumes: Contains resume versions/templates to be showcased
